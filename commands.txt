----------------------------------------------------------------------
Цей метод дозволяє автоматично оновити всі залежності у package.json:

Встановіть утиліту:
npm install -g npm-check-updates

Запустіть команду для перевірки оновлень:
npx npm-check-updates

Оновіть усі залежності до останніх версій:
npx npm-check-updates -u
npm install

----------------------------------------------------------------------
перевірка версії nuxt
npm list nuxt

----------------------------------------------------------------------
запуск каталога dist
npx serve dist

----------------------------------------------------------------------
запуск каталога .output/public
npx serve .output/public

----------------------------------------------------------------------
видалити node_modules
rm -rf node_modules

----------------------------------------------------------------------
npm ci (clean install) встановлює пакети чітко за версіями, вказаними у package-lock.json
npm ci

----------------------------------------------------------------------
глобально встановлює останню версію npm
npm install -g npm@latest
